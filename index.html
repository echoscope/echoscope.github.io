URL Notes:

URL scheme

scheme: defines message transmission scheme
hostname: remote server/computer name
domain: domain name or ip address
port: service port
path: file path
fileName file name

Absolute URL:
http://www.w3school.com.cn/images/picture.jpg
Relative URL:
/images/picture.jpg = [rootDirectory]/images/picture.jpg
images/picture.jpg = [currentDirectory]/images/picture.jpg
../images/picture.jpg = [parentDirectory]/images/picture.jpg

I had a Dream Today

今天我做了一个梦

之前做了好多梦，都是神奇的梦，体验丰富的梦。刚刚学会了 Markdown 语法，这种描写文本的方式真的是很优雅。想着就把这些梦境描述出来吧。

2020年6月21日
今天，梦见我何朋友旅游，到了一个像城固体育场一样的地方，但是又不是城固体育场，人很多，我记得我的面前是一栋酒店，酒店的面前有自动门。
我和朋友到了酒店，但是我跟他们没有住在同一个房间（是不是因为我在梦里也是如此不合群）。我记得我身边有刘飘。他陪我走进了电梯，电梯上去了，但是中途停了下来，我们走出去一看，这一层是群租房，群租房每个人都有一个隔间，最神奇的事情是这个群租房的环境很好，室温宜人，而且还有加湿器保障空气的湿润，群租房内还有一个火锅店，饿了随时可以进去吃火锅。我和刘都感叹，这个地方真好啊。不过我好像有事又出电梯了，到酒店外面了。她去哪里了，我不知道，可能去了她的酒店房间。于是我好像一个人又走进了电梯，我要去的是20层，但是我找了几遍才找到按钮，但是我没站稳，于是我按下了电梯24层。在电梯里，我看到电梯的不锈钢板上反射出来的金属光泽。于是我还突然问自己：这一定是梦，只不过是因为大脑对这种细节的还原很真实。
电梯往上走，到了24层，电梯把我送到了一个类似于工厂的车间一样的地方，电梯的地面还会自动移动，我根本不用走路。我进去了以后就像，我得再回电梯进去，不过我向右看了看，这个车间里面有深色水泥地，二十几米远的天花板上还有一根日光灯，灯下停着一辆仓库搬运车。车间里还有人，我觉得我必须赶紧走，以免被赶出去。
于是我又进了电梯。我好像是按到了20层，还是没按到，我忘记了。但是电梯一直往上升，潜意识告诉我这个电梯是智能电梯，就好像电影《心慌方》里面的立方体一样，它在建筑的三维结构里面寻找我要去的那个楼层，不过电梯一直飞速上升着，我抓紧左边的扶手杆。我发现电梯已经没有金属包裹了，呼啸的空气从我的面前飞过。电梯依旧往上走，我能以俯视的角度看见整个城市。这个城市是一个岛屿，四周都是水包围着，岛屿上有很多奇怪的建筑，还有一个最大的管状建筑，它就像一个废弃的弯形水管，在这个小小的岛屿上放着。我感觉这种视觉是我在现实生活里都没有体验过的。电梯到了顶层，它跨过顶层继续快速前进着。然后它沿着建筑的侧边快速运动……

HTML Notes

HTML site: https://whatwg.org/

HTML Introduction

HTML stands for Hypertext Markup Language.

Key points to understand HTML

  1. HTML is not a computer programming language.
  2. As a package of tags, HTML is used to describe text in a structured way.
  3. HTML structured text file usually has a .html filename suffix.
  4. What web browser, viewer, parser … do is to read HTML file and ouput the structured page so user can read joyfully.

Key points to understand HTML tag

  1. HTML is a pacakge of markup tags that can be used in plain text to construct HTML page.
  2. HTML tag usually appears in paris, first one is called opening tag and the second closing tag.
  3. HTML tag with their inner content is called a HTML Element. Example: <p> Content </p>
  4. Single HTML tag is called empty tag or empty element. Example: <br/>

DOCTYPE

As other software standards, HTML standards has its own development progress. The latest standard is HTML 5.
In every HTML Document, you would see a <!DOCTYPE> tag at the very beginning of the Document. The tag determints the HTML standard the document are using.
The HTML 5, will have a <!DOCTYPE html> as DOCTYPE.
For now there are still many web pages using HTML 4.0.1 and XHTML 1.0. Their Doctype are as follows:
HTML 4.0.1:
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
XHTML 1.0
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
