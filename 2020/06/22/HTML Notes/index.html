HTML Introduction

HTML stands for Hypertext Markup Language.

Key points to understand HTML

  1. HTML is not a computer programming language.
  2. As a package of tags, HTML is used to describe text in a form a structured way.
  3. HTML structured text file usually has a .html filename suffix.
  4. What web browser, viewer, parser … do is to read HTML file and ouput the structured page so user can read joyfully.

    Key points to understand HTML tag

  5. HTML is a pacakge of markup tags that can be used in plain text to construct HTML page.
  6. HTML tag usually appears in paris, first one is called opening tag and the second closing tag.
  7. HTML tag with their inner content is called a HTML Element. Example: <p> Content </p>
  8. Single HTML tag is called empty tag or empty element. Example: <br/>
