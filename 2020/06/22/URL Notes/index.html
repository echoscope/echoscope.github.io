URL Notes

URL Notes:

URL scheme

scheme: defines message transmission scheme
hostname: remote server/computer name
domain: domain name or ip address
port: service port
path: file path
fileName file name

Absolute URL:
http://www.w3school.com.cn/images/picture.jpg
Relative URL:
/images/picture.jpg = [rootDirectory]/images/picture.jpg
images/picture.jpg = [currentDirectory]/images/picture.jpg
../images/picture.jpg = [parentDirectory]/images/picture.jpg

URL Encode:
URL can only be stored or transmitted by using ASCII codes.
If must other than ASCII, then must use % to encode the character
